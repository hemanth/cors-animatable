# animatable-now 
> Check if a CSS property can be animated.


## Usage:

__Just access the URL with CSS property:__

```
https://animatable.now.sh/<CSS_PROPERTY>
```

__Examples:__

* https://animatable.now.sh/background

* https://animatable.now.sh/margine
